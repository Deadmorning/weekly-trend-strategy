# 策略开发思维框架：认知 → 信息 → 行为 → 机制 → 策略

> 本文档总结了从周期趋势策略开发过程中提炼出的完整思维框架，适用于后续所有量化策略的设计与评估。

---

## 框架概览

```
认知限制（持久，神经系统层面约束，无法被市场消除）
    +
信息结构（分散 vs 集中，决定偏差的传播方式）
    ↓
行为模式（集体行为，可以统计观测）
    ↓
市场机制（因果描述，描述性，不可直接使用）
    ↓  （需要：方向性 + 可观测信号 + 正期望值）
交易策略（规范性，可部署，有完整规则）
    ↓
寄生于特定周期阶段（无法逃离，只能选择寄生在哪个阶段）
```

**两个反馈回路（容易被忽略）：**

```
反馈①（反身性）：价格变化 → 成为新的信息 → 影响行为 → 进一步影响价格
反馈②（策略冲击）：策略被大量使用 → 对应机制被削弱 → Alpha 衰减
```

这两个回路决定了策略的持久性边界。

---

## 第一层：认知（Cognition）

这是最底层，也是所有市场规律的起点。

人类大脑的结构性限制在市场中主要表现为：

| 认知限制 | 具体表现 | 市场影响 |
|---------|---------|---------|
| 注意力有限（有界理性） | 不能同时处理所有信息，只关注显著变化 | 信息扩散有时序，产生注意力级联 |
| 损失厌恶 | 对亏损的痛苦远大于对同等盈利的喜悦 | 过早止盈、过久持亏 |
| 从众心理 | 在不确定情况下模仿他人行为 | 羊群效应、趋势自我强化 |
| 锚定效应 | 价格被近期高低点锚定，影响决策 | 突破前阻力、整数关口效应 |
| 处置效应 | 过早卖出盈利，过久持有亏损 | 上涨阶段卖压提前出现 |
| **过度自信** | 高估自己的判断能力，低估尾部风险 | 仓位过重、止损过松、频繁交易 |
| **近因偏差** | 过度加权近期事件，忽略长期基率 | 追涨杀跌；熊市末期极度悲观 |
| **确认偏误** | 主动寻找支持已有判断的信息 | 持有亏损时选择性忽略负面信号 |

**关键结论**：认知限制是生理和神经系统层面的约束，无法通过市场机制消除，因此是持久的。这是行为 Alpha 不会随策略使用人数增加而快速衰减的根本原因。

---

## 第二层：信息（Information）

信息结构决定了认知偏差如何在市场中传播和放大。

### 两种信息结构，对应两类偏差来源

**分散型信息结构（多数市场的常态）**
- 信息在参与者之间逐渐扩散
- 偏差来源是个体认知错误的集合
- 产生的行为：羊群、FOMO、恐慌抛售
- 策略方向：趋势跟踪、均值回归

**集中型信息结构（政策主导、权力集中市场）**
- 关键信息掌握在少数人手里
- 偏差来源从认知层转移到结构层
- 产生的行为：全市场追踪信息源、政策驱动的过度反应
- 典型案例：A股政策周期、当前美股政策集中化

### 反身性：价格本身是信息（重要补充）

信息不只是外部新闻和数据，**价格的变化本身会成为新的信息**，影响后续行为，形成自我强化的回路：

```
价格上涨 → 投资者将"上涨趋势"视为信息 → 买入决策 → 价格进一步上涨
```

这就是反身性（Reflexivity）。它的作用：
- **放大趋势**：在趋势中期，反身性使得趋势超过基本面支撑的水平
- **制造超调**：价格最终会偏离"应该"的位置，因为价格本身在改变参与者的行为
- **影响策略设计**：趋势策略在反身性强的阶段效果最好；均值回归策略在反身性反转后效果最好

> **关键区分**：分散型信息结构下，反身性由大量参与者的认知偏差共同驱动，相对可预测；集中型信息结构下，反身性由信息持有者的决策驱动，外部参与者基本无法建模。

### 市场成熟度与偏差的关系

> 市场成熟度降低的是认知偏差的影响，但不能消除信息不对称偏差。

| 市场 | 主要偏差来源 |
|------|------------|
| A股 | 散户占比高（认知偏差为主）+ 政策集中（信息不对称叠加） |
| 美股（当前） | 算法、机构主导的认知偏差已弱化，但政策不确定性带来新的信息不对称 |
| 港股 | 夹在A股政策周期和全球资本之间，信息来源多元但方向冲突 |

**关键结论**：行为偏差不会因为市场成熟而消失，它只会换来源。越成熟的市场，结构性信息不对称的影响反而越显眼。

## 第三层：行为（Behavior）

认知限制在特定信息环境下，产生可以观测的集体行为模式。

| 行为类型 | 认知根源 | 信息条件 | 市场表现 | 分类 |
|---------|---------|---------|---------|------|
| 羊群效应 / 恐慌 | 从众 + 损失厌恶 | 信息模糊、不确定 | 趋势延续、加速反转 | Type A |
| FOMO / 动量追逐 | 近因偏差 + 损失厌恶（错失） | 价格信号显著，反身性强 | 追涨杀跌 | Type B |
| 过度反应 | 注意力偏向显著事件 + 过度自信 | 单一事件冲击 | 短期过冲后回归 | Type C |
| 多周期盲点 | 注意力集中在单一时间框架 | 各时间维度信息不整合 | 短期噪声掩盖长期信号 | Type D |

> **注意**：注意力级联不是独立的行为类型，而是信息传播的**机制描述**（量先于价格变化）。它的产生根源是 Type A/B 的注意力分配有时序，本身不构成可独立部署的策略（原因：缺乏方向性）。

**个体行为 → 集体行为的聚合机制**：
- 个体认知偏差是独立发生的，但在相似信息环境下，方向往往一致
- 这种"方向一致"被反身性放大：价格朝一个方向动 → 强化了其他人做相同决策的信心
- 最终形成可以统计观测的集体行为模式

**关键结论**：行为是认知限制、信息结构、反身性共同作用的结果。

## 第四层：机制（Mechanism）

行为模式反复发生，在市场价格序列中形成可以统计观测的规律。

### 机制的定义

机制是对市场现象的因果描述，解释"为什么会发生"。它是**描述性的**，不是操作性的。机制不等于策略——将机制直接当作策略使用是策略设计中最常见的错误。

### 内容不可预测 vs 过程可预测

| 层面 | 可预测性 | 说明 |
|------|---------|------|
| 内容层（触发事件） | 不可预测 | 下一个重大新闻、政策转向、黑天鹅 |
| 过程层（参与者反应） | 有规律 | 触发后，参与者如何反应，行为模式可观测 |

机制利用的是过程可预测性，不是内容可预测性。市场"不可预测"和"行为有规律"并不矛盾——它们在不同层级上各自成立。

### 机制与周期的关系

> 行为偏差是周期的产生机制，周期是行为偏差的宏观结果。

两者是同一现象的不同描述角度，不可分离。**所有利用机制的策略，都必然寄生于某个周期阶段。**

| 周期阶段 | 主导行为类型 | 有效策略方向 |
|---------|------------|------------|
| 趋势上升段 | Type A（羊群延续）+ Type B（FOMO） | 趋势跟踪 |
| 趋势下降段 | Type A（恐慌）+ Type C（过度反应） | 趋势跟踪（空）或事件反转 |
| 震荡段 | Type C（过度反应短期） | 均值回归 |
| 周期转折点 | Type D（多周期盲点）+ 反身性反转 | 多周期框架 |

### 机制无法直接成为策略的条件

| 缺失条件 | 案例 |
|---------|------|
| 缺乏方向性 | 纯成交量信号：量大可能是买入也可能是恐慌卖出 |
| 信号代理质量差 | 宽基ETF成交量含申购赎回噪声 |
| 效果太小 | 扣除交易成本后 Alpha 消失 |
| 触发条件本身不可观测 | 需要区分是机构积累还是散户恐慌 |
| 信号无法领先价格 | 反身性已经把信号定价进去了 |

## Alpha 类型分类（核心概念）

策略 Alpha 的来源决定了它的持久性，在策略设计时必须明确。

| Alpha 类型 | 来源 | 持久性 | 衰减方式 |
|-----------|------|--------|---------|
| **行为 Alpha** | 利用认知偏差（损失厌恶、从众等） | 高 | 不随使用人数增加而衰减，因为认知偏差来自神经系统结构 |
| **信息 Alpha** | 利用信息不对称或更快的信息处理 | 低 | 使用者增多 → 信息被更快定价 → Alpha 消失 |
| **结构 Alpha** | 利用市场规则、制度或产品结构 | 中 | 随规则变化而变化，监管调整可使其突然消失 |

> **关键推论**：基于认知偏差的趋势跟踪策略（v2 类型）属于行为 Alpha，理论上不会因为使用人数增加而快速衰减——因为就算人人都知道这个策略，下次市场恐慌时散户的生理应激反应仍然会发生。

## 第五层：策略（Strategy）

将机制转化为完整可部署系统的过程。

### 转化路径

```
Step 1：明确利用的偏差
  └─ 哪一类市场参与者？哪种认知限制？哪种信息结构？
  └─ 这是行为 Alpha、信息 Alpha、还是结构 Alpha？

Step 2：找到可观测的代理信号
  └─ 这个偏差在价格/量/时间序列上如何表现？
  └─ 代理信号的质量如何？有没有噪声污染源？

Step 3：确认方向性
  └─ 趋势 / 均值回归 / 套利？
  └─ 任何缺乏方向性的机制，必须先确定方向层，再加入该机制

Step 4：先定义失败条件（在什么市场状态下，这个偏差消失？）
  └─ 失败条件写不出来 → 对机制理解不足，暂停开发
  └─ 区分两类失败（见下方"失效类型"）

Step 5：最小实现
  └─ 用最简单的方式表达机制，不要过早加复杂度
  └─ 每增加一个信号层，必须回答"这一层解决了哪个已定义的问题"

Step 6：独立验证（Walk-Forward，而非优化）
  └─ 验证的目标是找"在未来也成立的参数"，不是"历史最优参数"
  └─ 如果 WFA 失败，结论在测试前就要定好，而不是事后解释

Step 7：与同类策略对比
  └─ 是否真的在利用独立的偏差？
  └─ 是否比更简单的同类策略更好？（如果不是，退回 Step 5）
```

### 失效类型分类

在定义失败条件时，必须区分两类失效——它们有完全不同的应对方式：

| 失效类型 | 触发原因 | 是否会恢复 | 应对方式 |
|---------|---------|----------|---------|
| **周期性失效** | 当前处于策略不适用的市场阶段 | 是，下一个适用阶段恢复 | 暂停使用，等待市场阶段切换 |
| **机制衰减** | 该偏差正在被套利，越来越多人使用同类策略 | 缓慢衰减，不会突然恢复 | 监控 Alpha 是否在下降通道中 |
| **结构性失效** | 市场规则、产品结构、监管环境改变 | 可能永久失效 | 重新评估机制假设是否仍成立 |

### 策略设计原则

**1. 一个策略对应一个偏差**

混合多个偏差会让失效归因变得不可能，也增加过拟合风险。当策略表现变差时，你无法判断是哪个偏差失效了。

**2. 简单优于复杂**

从11策略对比实验中，越基础的策略越稳健——因为简单策略的过拟合空间小，参数少，边界清晰。三行逻辑能解释的 Alpha，不需要十个参数来实现。

**3. 先写失败条件，再写成功条件**

知道策略在哪里不该用，比知道它在哪里好用更重要。失败条件是策略的边界，也是部署时最重要的风控信号。

**4. 偏差类型决定策略的周期依赖**

- Type A（羊群/恐慌）策略 → 寄生于趋势阶段
- Type B（FOMO/动量）策略 → 寄生于牛市加速段
- Type C（过度反应）策略 → 寄生于高波动事件后
- Type D（多周期盲点）策略 → 全周期更稳健，但信号频率低

选择策略等于选择寄生的周期阶段，没有例外。

**5. 策略组合时考虑偏差相关性**

同类偏差的策略高度相关，在同一市场阶段会同时有效或同时失效。分散化应该在**偏差类型**上分散，而不只是在资产上分散：

```
Type A 策略 + Type C 策略 → 偏差类型对冲，周期依赖不同，组合更稳健
Type A 策略 + 另一个 Type A 策略 → 表面上两个策略，实质上是一个风险
```

## 策略监控原则（部署后）

策略上线不是终点。需要持续监控以区分"正常回撤"和"机制失效信号"：

| 监控指标 | 正常回撤信号 | 机制失效信号 |
|---------|------------|------------|
| 回撤时间 | 与历史最大回撤周期相近 | 持续时间显著超过历史 |
| 回撤来源 | 与周期阶段匹配（震荡期回撤） | 在适用的市场阶段也表现差 |
| Alpha 趋势 | 年度 Alpha 在均值附近波动 | Alpha 连续多年单方向下降 |
| 同类对比 | 同偏差类型的其他策略也回撤 | 只有本策略表现差，同类策略仍有效 |

> **核心判断逻辑**：如果市场处于该策略应该有效的阶段，但策略仍然失效 → 这是机制失效信号，不是暂时回撤。

## 三个核心问题（策略开发前必须回答）

在开始写第一行代码之前，先回答：

**Q1：我在利用哪一层的哪类偏差，Alpha 来自哪里？**

是认知层（行为 Alpha）、信息层（信息 Alpha），还是结构层（结构 Alpha）？这决定了策略在什么市场结构下有效，以及 Alpha 的持久性和衰减方式。

**Q2：这个机制能独立转化为策略吗？**

它是否有方向性？代理信号质量如何？扣除成本后还有 Alpha 吗？有没有反身性或信息价格发现导致信号已经被市场定价？

**Q3：这个偏差在什么时候消失，是暂时性还是结构性？**

先写失败条件，并明确它属于哪种失效类型：
- 周期性失效（市场阶段不对，会恢复）→ 暂停策略
- 机制衰减（Alpha 缓慢下降通道）→ 降低仓位，追踪趋势
- 结构性失效（市场规则改变）→ 重新评估机制假设

## 本项目策略（v2）的框架定位

| 维度 | v2 策略的答案 |
|------|-------------|
| 利用的偏差 | Type A（羊群/恐慌）+ Type D（多周期盲点） |
| Alpha 类型 | 行为 Alpha（纪律） |
| 信息结构假设 | 分散型，散户占主导，反身性驱动趋势延续 |
| 入场依据 | 趋势确立后（价格已在趋势中），利用羊群效应和反身性延续 |
| 方向性 | 明确：只做多，不做空 |
| 周期依赖 | 寄生于趋势阶段；震荡/快速轮动市场效果差 |
| 失败条件（周期性） | 高波动、快速轮动、无趋势市场阶段 |
| 失败条件（结构性） | 市场散户占比大幅下降；算法交易将反应时间压缩到日线以下 |
| Alpha 持久性 | 高——认知偏差来自神经系统结构，不随策略使用人数增加而快速衰减 |
