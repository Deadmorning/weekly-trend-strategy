LAYER 0 - Alpha vs Beta:

本策略是主动择时策略，基准应为沪深300ETF的买入持有（Buy-and-Hold）。

策略目标：通过均线交叉信号识别趋势启动和结束时机，从而在上涨趋势中持有，在下跌趋势中空仓，以期规避大幅回撤的同时捕获主要上涨行情。

是否是Alpha？部分是。判断如下：
- 如果仅在多头市场中捕获涨幅、空仓期持有现金，在震荡市中会因频繁交叉而连续止损，最终跑输买入持有——这是**负Alpha**。
- 如果有效规避了重大熊市（2008、2015、2022），总体收益/回撤比优于买入持有——这才是真Alpha。
- 均线交叉本身是**趋势跟随**逻辑，Alpha来源于能否在趋势持续阶段坚持持仓，而不是在震荡阶段减少交易成本。

结论：这是一个**带条件的Alpha策略**。能否产生Alpha，取决于沪深300的趋势特征强度——后续各层分析。


LAYER 1+2 - Mechanism Source:

机制来源：**行为路径（Behavioral Path）**

具体路径如下：

```
[认知限制：羊群效应 + 追涨杀跌心理] × [信息结构：分散型，价格本身成为主要信号]
    ↓
集体行为规律：趋势一旦确立，越来越多参与者跟进入场（正反馈）
    ↓
市场机制：趋势在中段自我强化（价格反身性），形成可识别的持续动量
    ↓
策略：用均线交叉作为趋势已确立的滞后确认信号，入场跟随趋势
```

**反身性（Reflexivity）的作用：**
沪深300以散户和趋势型机构资金为主，价格上涨本身吸引新资金进入，进一步推涨价格。这是均线交叉策略在A股有效的结构性原因——趋势不仅存在，而且因反身性会持续足够长的时间，给滞后信号留出盈利窗口。

**基本面辅助：**
ETF跟踪宽基指数，基本面渐进修正（经济数据、政策预期的逐步变化）也会在价格趋势中体现，增强趋势持续性。


LAYER 3 - Bias Type:

**主要类型：Type A（羊群效应 + 恐慌）→ 趋势跟随**

- 上涨时：羊群效应推动资金持续流入，形成正反馈趋势
- 下跌时：恐慌情绪导致集体减仓，趋势加速
- 均线交叉是趋势**已经确立后的滞后确认**，不是预测——这符合Type A策略的设计逻辑

**次要类型：Type B（FOMO + 动量）**

5日/20日均线参数较短，在短期动量效应上也有部分捕获，与Type B有重叠。

风险评估：主次类型明确，重叠度可接受，但需要注意参数较短带来的噪音——5日MA对短期噪声敏感，会产生频繁假信号，尤其在Type A趋势尚未形成时误判Type B的短期动量。

  - 唯一风险：节假日休市后的跳空可能导致信号执行价与理论信号价存在偏差，属于可管理的滑点问题

FAILURE CONDITIONS:

**Cyclical（周期性失效）：**
  - 触发条件：市场进入横盘震荡区间（如2012-2014、2022下半年-2023上半年），无明显趋势
  - 识别方式：连续出现3次以上"金叉后7个交易日内死叉"的情况（快速震荡假信号）
  - 响应方式：暂停交易，保持现金，等待趋势重新确立（而非修改参数去适应）
  - 预期恢复：下一个趋势行情启动时自然恢复

**Mechanism Decay（机制衰减）：**
  - 触发条件：均线交叉后的平均持仓盈利持续缩短（例如从平均持仓15天盈利缩短至5天）
  - 这是大量量化资金开始在均线交叉前预判并提前买入/卖出的信号
  - 响应方式：逐步缩减仓位，监测趋势。注意：沪深300ETF体量大、趋势时间长，这种衰减在宽基指数上比个股慢得多

**Structural（结构性失效）：**
  - 触发条件之一：沪深300指数编制方法重大变化，导致其历史趋势特征与未来分离
  - 触发条件之二：ETF申购赎回机制变化，导致场内ETF价格与净值持续偏离，产生系统性价格失真
  - 触发条件之三：交易制度变化（如T+0改为T+2），改变持仓成本结构
  - 响应方式：重新评估机制假设，不能假设策略可以自然恢复
